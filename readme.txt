
# PROJETO INTEGRADOR

Este repositório contém o código-fonte do projeto integrador desenvolvido durante o II EMI em Informática no IFTO. O objetivo principal do projeto é integrar conceitos e habilidades adquiridos nas disciplinas de Introdução a Banco de Dados, Linguagem de Script, Programação Dinâmica para WEB e Instalação e Configuração de Servidores.


**Objetivo**

Desenvolver um protótipo de um sistema web para a Progressão Funcional do Docente, baseado no formulário e pontuação das atividades de ensino descritas no Apêndice B. O sistema deve permitir que os docentes cadastrem informações relevantes, submetam documentos comprobatórios e finalizem o preenchimento para revisão pelo setor responsável (CPPD).
